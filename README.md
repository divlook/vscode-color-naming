# Color Naming

A VS Code Extension that names colors using ntc.js.

## Features

\!\[feature X\]\(images/feature-x.png\)

> 스크린샷 준비 중

## Extension Settings

- `"colorNaming.caseStyle": "Kebab"`

    NAME            | DESC
    ----------------|------------------------------
    Kebab (default) | ex) kebab-case, light-gray-10
    Camel           | ex) camelCase, lightGray_10
